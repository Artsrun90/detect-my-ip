Detección de Dirección IP

Nuestro servicio detecta automáticamente tu dirección IP pública al cargar la página. No se requieren acciones adicionales.

¿Qué muestra la dirección IP?

  • Tu dirección IP pública (IPv4 y/o IPv6)
  • Tipo de conexión (estática/dinámica)
  • Información del proveedor

Cómo interpretar los resultados

La dirección IP se muestra en dos formatos:

  • IPv4: Cuatro números del 0 al 255 (ejemplo: 192.168.1.1)
  • IPv6: Ocho grupos de dígitos hexadecimales (ejemplo: 2001:0db8:85a3::8a2e:0370:7334)

Prueba de Velocidad de Internet

Nuestra prueba de velocidad mide la velocidad de descarga y ping de tu conexión.

Cómo ejecutar la prueba

  1. Haz clic en el botón "Iniciar Prueba" en el bloque Speed Test
  2. Espera a que termine la prueba de ping
  3. Espera a que termine la prueba de velocidad de descarga
  4. Revisa los resultados

Recomendaciones importantes

  • Cierra todas las aplicaciones innecesarias
  • No descargues archivos durante la prueba
  • Usa conexión por cable para mayor precisión
  • Repite la prueba varias veces para obtener un promedio

Entender los resultados

Parámetro Descripción Buenos valores
Velocidad de descarga Velocidad de recepción de datos > 25 Mbps
Ping Tiempo de respuesta del servidor < 50 ms

Geolocalización por IP

Determinamos la ubicación aproximada de tu dirección IP basándonos en una base de datos de geolocalización.

Precisión de la geolocalización

La precisión puede variar dependiendo de:

  • Tipo de tu conexión
  • Uso de VPN o proxy
  • Ubicación de los servidores de tu proveedor

¿Qué muestra la geolocalización?

  • País y región
  • Ciudad (aproximadamente)
  • Coordenadas (latitud/longitud)
  • Zona horaria

Herramientas DNS

Nuestras herramientas DNS ayudan a analizar nombres de dominio y sus registros.

Herramientas disponibles

  • DNS Lookup: Búsqueda de registros de dominio
  • Reverse DNS: Búsqueda de dominio por IP
  • WHOIS: Información de registro de dominio

Cómo usar

  1. Ingresa el nombre de dominio o dirección IP
  2. Selecciona el tipo de registro (A, AAAA, MX, CNAME)
  3. Haz clic en "Buscar"
  4. Revisa los resultados

Verificación de Seguridad

Verificamos tu dirección IP en busca de listas negras y actividad sospechosa.

¿Qué se verifica?

  • Presencia en listas de spam
  • Bloqueo por sistemas antivirus
  • Actividad sospechosa
  • Uso de VPN/proxy

Si la IP está bloqueada

Contacta a tu proveedor de internet para resolver el problema o usa una VPN para cambiar tu dirección IP.

Solución de Problemas

Problemas comunes y soluciones

No se muestra la dirección IP

Causa: Problemas con JavaScript o bloqueo de publicidad

Solución: Desactiva bloqueadores de publicidad, actualiza la página

Geolocalización incorrecta

Causa: Uso de VPN, proxy o internet móvil

Solución: Desactiva VPN, usa internet doméstico

Prueba de velocidad lenta

Causa: Conexión lenta o carga del servidor

Solución: Intenta más tarde, verifica otras aplicaciones

Obtener ayuda

Si los problemas persisten, contáctanos a través de nuestra página de contacto.
